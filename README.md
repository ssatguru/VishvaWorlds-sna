# VishvaWorlds-sna
sensor and actuator demo

A Vishva world to demo various sensors and actuators  
see  
https://ssatguru.github.io/VishvaWorlds-sna/bin


To use  
clone this repository to some folder on your pc  
cd to that folder  
run some static http server ( I use nodejs http-server)  
open browser and goto http://localhost:8080/bin  (assuming you started the server on port 8080)  
make changes  
to save, click the download scene button and when promted, do save link as and save to visha/worlds/sna.json 
overwriting the  previous sna.json
